# Expenses Tracker App

A simple and intuitive expense tracking application built with vanilla JavaScript and HTML.

## Features

- ✅ Add and track expenses
- ✅ Categorize spending
- ✅ View total expenses
- ✅ Delete expense entries
- ✅ Clean, responsive UI

## Tech Stack

- **Frontend**: HTML5, CSS3 (Tailwind CSS)
- **JavaScript**: Vanilla JS with array methods and DOM manipulation
- **Storage**: LocalStorage for data persistence

## Getting Started

1. Clone the repository
2. Open `index.html` in your browser
3. Start adding expenses!

## How to Use

1. Enter an expense amount
2. Add a description (optional)
3. Select a category
4. Click "Add Expense"
5. View your expenses in the list below
6. Click delete to remove entries
